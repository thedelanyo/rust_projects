# 🦀 Rust Practice Programs

A collection of small to intermediate Rust programs built while learning and mastering core Rust concepts.

This repository focuses on writing clean, idiomatic Rust code while solving practical problems involving user input, data structures, and control flow.


---

## 🛠️ Getting Started

### Prerequisites

- [Install Rust](https://rustup.rs)
- Clone the Repository

`git clone https://github.com/thedelanyo/rust-practice.git`
`cd rust-practice`

---
## 🎯 Goal

The goal of this repository is to:

- Build a strong foundation in Rust
- Practice writing safe and efficient code
- Transition from beginner to intermediate level
- Prepare for real-world backend and systems programming

---
